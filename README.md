# easy-choose

### It can easily help you choose!

<br>

## Installation
Use pip:
<br>
> `pip install easy-choose`

<br>

## Usage
It's particularly simple to use. The code is as follows:
> `>>>import easy_choose as ec`
<br>
> `>>>ec.choose(2, "A", "B", "C")`
<br>
> `["A", "C"]`
<br>

Of course, you can also use lists:
> `>>> ls = [1, 2, 3, 4, 5, 6, 7, 8, 9, 0]`
<br>
> `>>> ec.choose(5, *ls)`
<br>
> `[1, 4, 2, 7, 5]`
